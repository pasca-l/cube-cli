# Cube CLI
This product holds scripts to show a spinning cube on a terminal.

## Usage
- script written in python
```bash
$ python python/main.py
```

## Algorithm
- rotation is calculated by the [rotation matrix](https://en.wikipedia.org/wiki/Rotation_matrix)
